# Pizza-API
## Description

This project is designed to make API calls on our pizza API app.

## Getting Started

### Prerequisites

* DataBase Engine: MySql
* IDE used: IntelliJ
* Package manager: (npm)

### Configuration

#### How to set up the database ?
Execute the `Create_table.sql`

## Deployment
1. run `npm install`
2. run `npm run dev`
